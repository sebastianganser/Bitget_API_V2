# SDK | Bitget API

## SDK

We support below languages

| Language | Code Path                         |
|---------|------------------------------------|
| Java    | Check package `com.bitget.openapi.api.v2` |
| Python  | Check `v2`                         |
| NodeJs  | Check `src/lib/v2`                |
| Golang  | Check `pkg/client/v2`             |
| PHP     | Check `src/api/v2`                |
